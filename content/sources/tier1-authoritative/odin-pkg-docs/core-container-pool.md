## Types

### [Pool ¶](#Pool) [*Source*](https://github.com/odin-lang/Odin/tree/master/core/container/pool/pool.odin#L27)

```
Pool :: struct($T: typeid) {
	… // See source for fields
}
```

 

A thread-safe (between init and destroy) object pool backed by virtual growing arena returning stable pointers.
The element type requires an intrusive link node.

**Example:**

```
Elem :: struct {
	link: ^Elem,
}

p: pool.Pool(Elem)
pool.init(&p, "link")
```

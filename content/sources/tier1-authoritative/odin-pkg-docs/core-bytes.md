Buffer :: struct {
	buf:       [dynamic]u8,
	off:       int,
	last_read: Read_Op,
}

The zero value for Buffer is an empty buffer ready to use.

### [alias\_inexactly ¶](#alias_inexactly) [*Source*](https://github.com/odin-lang/Odin/tree/master/core/bytes/bytes.odin#L1481)

```
alias_inexactly :: proc "contextless" (a, b: []u8) -> bool {…}
```

 

alias\_inexactly returns true iff a and b have a non-zero length,
the base pointer of a and b are NOT equal, and any part of a overlaps
with b (ie: `alias(a, b)` with an exception that returns false for
`a == b`, `b = a[:len(a)-69]` and similar conditions).

fields\_proc makes no guarantee about the order in which it calls f(ch)
it assumes that `f` always returns the same value for a given ch

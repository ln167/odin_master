### [Log\_Allocator ¶](#Log_Allocator) [*Source*](https://github.com/odin-lang/Odin/tree/master/core/log/log_allocator.odin#L17)

```
Log_Allocator :: struct {
	allocator: runtime.Allocator,
	level:     runtime.Logger_Level,
	prefix:    string,
	lock:      sync.Mutex,
	size_fmt:  Log_Allocator_Format,
}
```

 

Log\_Allocator is an allocator which calls `context.logger` on each of its allocations operations.
The format can be changed by setting the `size_fmt: Log_Allocator_Format` field to either `Bytes` or `Human`.
